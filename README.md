# PasswordAnalyser

PasswordAnalyser est un programme qui permet de voir si le mot de passe est robuste.
Il vérifie que le mot de passe comprend au moins un chiffre, une majuscule, une minuscule, un caractère spécial.  
Il vérifie aussi qu'il ne contient pas de mot du dictionnaire français ainsi que de suite logique.  


Il fait un calcul d'entropie pour juger de la force mathématique du mot de passe, a l'aide de cette formule: 
E=L×log2(R)
